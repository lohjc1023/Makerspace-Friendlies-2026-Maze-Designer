# Makerspace-Friendlies-2026-Maze-Designer
A simple tool of HTML file for designing maze route of Makerspace Friendlies 2026 Competition. Maze features include Start Zone, End Zone, Free-Track Zone, Entrance Gateway, Walls, Checkpoints, Balls, Following Lines.
